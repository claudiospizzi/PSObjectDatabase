
TOPIC

SHORT DESCRIPTION
    PowerShell module to simply store, retrieve and query PowerShell objects in
    a local database abstracting the underlying storage technology.

LONG DESCRIPTION
    To get more information about this module, check the open source GitHub
    repository readme or the cmdlet based help for the module functions.
